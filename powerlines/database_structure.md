# Описание структуры базы данных

## Назначение

База поддерживает учёт воздушных линий электропередачи, опор, траверс, проводов и тросов, изоляторов, документации, замеров тока и отключений. Справочники (страны, регионы, материалы, марки проводников) вынесены отдельно для нормализации.

## Домены

- `support_kind` — тип опоры (промежуточная / анкерная).
- `conductor_kind` — провод или трос.
- `document_kind` — вид документа (проект, паспорт, протокол, прочее).
- `insulator_type` — тип изолятора (подвесной ряд, натяжной, штыревой, прочее).

## Таблицы

| Таблица | Назначение |
|--------|------------|
| `countries`, `regions` | Географическая иерархия для привязки ВЛ к территории |
| `materials` | Материалы конструкций (опора, изолятор, проводник, трос и т.д.) |
| `power_lines` | ВЛ: напряжение, длина, описание, дополнительные атрибуты в JSONB |
| `supports` | Опоры: тип, шифр; точки `base_geom`, `vertex_geom` — PostGIS `geometry(Point, 4326)` |
| `traverses` | Траверсы на опоре: материал, концы, угол разворота |
| `conductor_brands` | Справочник марок проводников (сечение, связь с материалом) |
| `conductors` | Пролёты: марка из `conductor_brands`, опоры, траверсы на концах, геометрия, стрела провеса |
| `insulators` | Изоляторы на опоре; тип задаётся доменом `insulator_type` |
| `documents` | Метаданные; `doc_kind` — домен `document_kind`, поле `meta` — JSONB |
| `documents_power_lines`, `documents_supports`, `documents_conductors` | Связи документов с сущностями (многие ко многим) |
| `line_current_measurements` | Замеры тока по ВЛ |
| `outages` | Отключения с интервалом времени и причиной |
| `audit_log` | Журнал событий для аудита |

Геоданные задаются расширением **PostGIS** (`CREATE EXTENSION postgis` в скрипте схемы), система координат **EPSG:4326** (WGS84).

## Связи

- Регион принадлежит стране; ВЛ принадлежит региону.
- Опора принадлежит одной ВЛ; траверсы и изоляторы — опоре; у траверсы указан материал.
- Проводник ссылается на марку (`conductor_brands`), две опоры и две траверсы на этих опорах; целостность проверяется триггером.
- Документы связаны с линиями, опорами и проводниками через отдельные таблицы связей.
